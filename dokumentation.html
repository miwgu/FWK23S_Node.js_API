Server Uppgift 2: Dokumentation av det här APIet

Routes

  • 🔐GET /users - returnerar alla användare.
  • 🔐GET /users/{id} - returnerar en user med angivet id eller status 204 om användaren saknas.
  • 🔐POST /users - Skapar en ny användare accepterar JSON object på formatet
    {"username":"unik namn", "first_name":"", "last_name":""}. username är obligatoriskt och ska var unikt.
  • POST /login - för inloggnin. Retunerar en JWT som används som bearer token i anrop till outes skyddade med auth.
    Accepterar JSON object på formatet {"username":"", "password":"", "last_name":""}

🔐= denna route kräver ett giltigt bearer token i authorization header.
